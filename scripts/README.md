# Scripts

Utilities developed to automate my C++ workflow.

## new_cpp_project

Creates a complete CMake-based Modern C++ project.

It automatically generates:

- README.md
- CMakeLists.txt
- .gitignore
- src/main.cpp
- build/
- include/
- docs/
- tests/

Installation:

```bash
cp new_cpp_project ~/.local/bin/
chmod +x ~/.local/bin/new_cpp_project
