¿qué significa ChatT, Traits y Allocator?

En c++ std::string no es un tipo unico y cerrado, en realidad es una plantilla o molde que se adapta a diferentes tipos de letras.

ChatT(character type): representa el tipo de caracter a usar.
    hay un caracter que se llama char, es el mas comun.
    ChatT es simplemente un marcador posicional que te dice "aqui va el tipo de letra que elijas"

Traits y Allocator: Son configuraciones avanzdas que le dicen al programa como comparar esas letras(Traits) y cómo manejar la memoria de la computadora (Allocator). Casi nunca tendras que tocarlos ni escribir nada para ellos, c++ los llena con valores por defecto de forma automatica

## Operadores aritmeticos:

int a = 10, b = 20, c = 0; 
a = b; // assigns b value to a value
a = b + 60; // adds 60 to b value, then assigns that value to a
a += 60; // same as previous
a = 2*b; // multiplication. Here same as a *= b;
a = b/3; // division, notice truncation for integers. a /= 3;
a = b%7; // modulo operator, residue
a = 10;
b = 20; 
a = b++; // post increment
a = 10;
b = 20; 
a = ++b; // pre increment
