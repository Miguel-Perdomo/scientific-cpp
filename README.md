# Scientific C++

A personal repository documenting my journey in modern C++ for scientific computing.

The primary goal of this repository is to strengthen my C++ foundations, software engineering practices, and build system knowledge before returning to large-scale scientific frameworks such as Geant4.

This repository is organized as a long-term learning project rather than a collection of finished applications.

---

## Repository structure

```
scientific-cpp/
│
├── learncpp/          # Exercises following LearnCpp
├── mini_projects/     # Small standalone C++ projects
├── design_patterns/   # Design pattern implementations
├── algorithms/        # STL algorithms and programming exercises
├── data_structures/   # Data structures implemented in C++
├── cmake/             # CMake examples and build configurations
├── playground/        # Quick experiments and prototypes
├── scripts/           # Bash utilities for development workflow
```

---

## Learning objectives

This repository focuses on developing practical experience with

- Modern C++ (C++17)
- Object-Oriented Programming
- Generic Programming
- Memory Management
- Templates
- STL
- Design Patterns
- CMake
- Git
- Linux development workflow

The long-term objective is to build a solid foundation for scientific software development, including simulation frameworks such as Geant4 and large research codebases.

This repository is intended to document the learning process.
