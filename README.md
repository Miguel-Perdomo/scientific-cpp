# Scientific C++

A collection of modern C++ projects focused on scientific computing, simulation and physics.

## Repository structure

- `learncpp/` — Exercises following LearnCpp.
- `mini_projects/` — Personal C++ projects.
- `playground/` — Small experiments and prototypes.
- `albedo/` — Development of the ALBEDO Monte Carlo project.
- `geant4/` — Geant4 examples and applications.
- `scripts/` — Utilities to automate my C++ workflow.

## Toolchain

- C++17
- GCC
- CMake
- Git
- Bash
- VS Code
- Linux / WSL2

This repository documents my journey toward scientific software development.
